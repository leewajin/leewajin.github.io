1. 한피스

- 해당 코드는 원피스 세계관에서 크로커다일과 루피가 싸우는 프로그램으로 항해를 하면서 스탯을 추가하고, 보물상자도 발견하고 전투 후 보상을 받는 등의 코드를 작성하였다.

2. 오픈소스 라이선스 식별 및 충돌 탐지 기술 개발(진행중)
- 기간 : 05. 21. ~ 08. 31. (3개월)
- 개요 : 오픈소스 소프트웨어의 보안 위협 증가로 인해 라이선스 문제가 늘어나면서 서로 충돌하는 요구사항을 가진 라이선스들로 인한 문제가 발생할 수 있기 때문에 이러한 충돌 문제를 자동으로 식별할 수 있는 기술 개발이 필요하다.
- 기술 방법 : SPDX 사이트에 라이선스 정보들이 올라와있는데 각 라이선스들의 주석문이 다양한 형태로 작성되어있는 것을 정규식을 통해 식별해야한다.
- 개발 언어 : 파이썬

- 외에도 취미, 여행간것 등 블로그처럼 내가 원하는 디자인으로 나만의 홈페이지를 만드는 것이 목표이다.
- 40-50살이 될때까지도 업데이트하는 홈페이지를 만들고싶다.

희망직무

정보보안전문가
● 직무 개요
- 조직의 정보 시스템을 보호하고, 데이터 무결성, 기밀성, 가용성을 유지하는 역할을 담당합니다. 사이버 공격, 데이터 유출, 해킹 등 다양한 위협으로부터 시스템을 보호하고, 보안 정책을 수립하며, 보안 솔루션을 구현하는 것이 주된 업무입니다.

● 주요 역할과 책임
1. 위협 분석 및 대응
- 사이버 위협을 분석하고 보안 취약점을 식별한다.
- 보안 사고 발생 시 신속히 대응하고, 사고를 최고화하기 위해 노력한다.

● 보안 정책 수립 및 시행
- 조직의 보안 정책을 수립하고, 회사에서 이를 준수할 수 있도록 교육 및 가이드를 제공합니다.
- 보안 규정을 지속적으로 업데이트하고, 직원들이 이를 준수하도록 관리한다.

● 보안 시스템 구축 및 관리
- 방화벽, 침입 탑지 시스템, 침입 방지 시스템 등의 보안 솔루션을 설치하고 관리한다.
- 네트워크와 시스템 보안 설정을 강화하여 외부 위협을 차단한다.

● 필요한 자격
- 기술적 역량 : 방화벽, VPN, 운영체제 보안 설정, 서버 보안, 애플리케이션 보안, 데이터 암호화, 전송 암호화 등의 대한 이해
- 자격증 : 정보처리기사, 정보보안기사, 리눅스 마스터 등
